Introduction

Welcome! The purpose of this manual is to take common data structures and algorithms and explain them clearly, with code implementations in Java along the way. Data structures and algorithms are used to build software of all kinds.

Who is this for?

This manual is for software engineers or anyone interested in understanding data structures and algorithms better.

What can I expect?

As the title suggests, the manual is broken down into two main sections: Data Structures and Algorithms. Data Structures gives an introduction to data structures, assuming no prior knowledge or experience except for reasonable experience with the Java programming language. Algorithms introduces algorithms and explains them in detail.

Before diving into data structures and algorithms, though, there's a brief primer on computational complexity, which is used to analyse the efficiency of various data structures and algorithms. Feel free to use the navigation menu to explore the manual in any order you like. Enjoy!
